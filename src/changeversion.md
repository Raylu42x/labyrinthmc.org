# How To Change Your Java Game Version
Note: this only works for Java Edition

## Step 1: Find Installations
Click on the installations tab in minecraft launcher
<img src="/assets/img/changeVersion1.png" alt="Installations tab" class="change-version-img">

## Step 2: New Installation
Click new installation towards the top of the window
<img src="/assets/img/changeVersion2.png" alt="New installation button" class="change-version-img">

## Step 3: Open Versions
Click the version dropdown. Also fill in the name 
<img src="/assets/img/changeVersion3.png" alt="Open versions dropdown" class="change-version-img">

## Step 4: Choose Game Version
Choose the right release version for the server's version. Do not choose a snapshot
<img src="/assets/img/changeVersion4.png" alt="Choose game version" class="change-version-img">

## Step 5: Finish and Install
Click the install button in the bottom right
<img src="/assets/img/changeVersion5.png" alt="Finish and install" class="change-version-img">
