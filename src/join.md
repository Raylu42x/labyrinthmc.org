# How to Join

The server is **whitelisted**. Contact an admin to get added before joining.
  
To get whitelisted, email [admin@labyrinthmc.org](mailto:admin@labyrinthmc.org) or find Bennett at Labyrinth.

---

### Switching To The New IP
Remove the old server address/server or edit it, put in the address and port for your system, see down

## Bedrock Edition

The server runs [Geyser](https://geysermc.org/) and [Floodgate](https://github.com/GeyserMC/Floodgate), so Bedrock
players can join too.

| | |
|---|---|                              
| **Server Address** | `bedrock.labyrinthmc.org` |
| **Port** | `58543` |

### PC & iPad
  
Enter the server address and port above in the **Add Server** screen.

### Consoles
  
There are several ways:                
 - **BedrockTogether** — an iOS/Android app that makes any server appear as a LAN game
 - **Geyser DNS** — may work depending on your platform; still being tested with the new Bedrock UI
